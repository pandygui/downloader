# Deepin downloader

Deepin Downloader is a built-in downloading tool of deepin operating system, supporting single and batch download.

## Dependencies

* sudo apt install libdtkwidget-dev libqt5svg5-dev qttools5-dev-tools

## Installation

* mkdir build
* cd build
* qmake ..
* make

## Usage

* ./deepin-downloader
